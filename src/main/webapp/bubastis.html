An Ontology Diff Tool (version 1.1)

Bubastis is a tool for detecting asserted logical differences between two ontologies, such as between versions. A stand alone version of the tool and a Bubastis Java library is also available for download from the Bubastis GitHub repository. Bubastis is powered by the OWL-API.
